企画

ニーズ評価

評価 制作チーム

最初の会議の後、プロジェクトの要件を決定します。

  • assume target audience
  • assume current customer demographics and buying habits
  • SWOT Analysis - Strengths, Weaknesses, Opportunities and Threats
  • define unique value proposition
  • define goals (ie. selling bikes) and campaign objectives

  • Ordering to meet goals - example (of selling bikes goal) below:

    1. The newest model bike
    2. The best-selling bike
    3. “Find your perfect ride” CTA
    4. Company name and hero image
    5. Navigation
    6. Search
    7. The second-best-selling bike
    8. Gift certificates
    9. A testimonial
    10. The latest blog post

レビュー お客様

プロジェクトの要件が先方様と確認して、修正してます。

- 2週目 -

リサーチ 制作チーム

時間をかけて最良の解決策とアプローチを研究します。コンテンツインベントリを使って、目的のために順番を決めてください。

  • research target audience and competitors
  • research prospective customers if part of goals
  • outline creative idea
  • define objective and message
  • propose an updated budget (provide quote) and timeline
  • determine domain / server / mail server / new mail accounts needed

ワイヤーフレーム

ワイヤーフレーム 制作チーム

PCとSP用の白黒ワイヤーを作っています。SPでコンテンツが優先通りで順番して、絶対必要物だけ入れて、必要ない物が下層ページだけで良いです。

  • create a visual example of this using the bikes example to remove second best seller and gift certificates.
  • tablet users are too few to focus extensively on, so tablet size (768px~1024px) fixes to side margins and font sizes can be done via a separate media query after creating SP/PC versions.
